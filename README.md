Project Description:

Wiggle Wiggle Academy is a whimsical, underwater-themed web application dedicated to the noble and hilarious mission of teaching baby fish how to swim. It's a dynamic, full-stack platform that parodies modern e-learning systems, tailored entirely for our aquatic friends.

The Problem (that doesn't exist):

Across the vast, uncharted oceans, countless baby fish are wiggling aimlessly. Their potential is untapped, their technique is sloppy, and their parents are worried. There is a clear and present lack of a structured, credential-based educational system to turn these amateur floaters into professional swimmers.

The Solution (that nobody asked for):

Wiggle Wiggle Academy is the world's premier digital institution for aquatic locomotion. We provide a robust online platform where fish trainees can log in, receive a personalized swim kit, attend video tutorials, and be tested with a practical "Obstacle Fin-Course" exam. It's an over-engineered, completely useless solution to a problem that never existed.

Implementation

For Software:
Implementation 
The Wiggle wiggle academy project is implemented using the Flask web framework for the backend and HTML/CSS/JavaScript for the frontend.

Backend Implementation:

Developed using Python (Flask) to handle user authentication (login, registration) and routing.

Routes:

/register ‚Üí Handles new user registration.

/login ‚Üí Validates user credentials and redirects to the home page.

/home ‚Üí Displays the main dashboard after login.

Flask session management used to maintain user state.

Frontend Implementation:

HTML pages (login.html, register.html, home.html) styled with CSS for a playful, underwater theme.

Background videos implemented using the <video> HTML tag for dynamic visuals.

JavaScript used for form validation and interactive UI elements.

Installation:

Clone the repository to  local machine.

Ensure we have Python 3 and pip installed.

Create a users.txt file with the following content:
Flask

Install the necessary packages by running:
pip install -r requirements.txt

Run

Navigate to the root directory of the project in your terminal.

Run the Flask application using the following command:
python login.py

Open your web browser and go to http://127.0.0.1:5000/ to see the application live.
